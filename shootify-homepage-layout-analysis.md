# Shootify 首页空间布局与间距分析

> 基于页面截图 (1440px 视口宽度) 与 HTML 源码结构综合整理
> 页面总高度: ~12235px | 技术栈: WordPress 自定义主题 + Wonder Blocks (Elementor)

---

## 全局设计参数

| 参数 | 值 |
|------|-----|
| **视口基准宽度** | 1440px (桌面端) |
| **内容最大宽度** | 约 1200-1280px (居中容器) |
| **全局左右内边距** | 约 20-40px (移动端递增) |
| **主色调** | #000000 黑色 / #FFFFFF 白色 |
| **强调色/CTA** | 黄绿色 ~#D4E157 或类似亮黄绿 |
| **辅助背景色** | #F8F9FA 或 #FAFAFA 浅灰交替 |
| **字体系统** | 无衬线体 (Inter/SF Pro 风格) |
| **圆角风格** | 小圆角 4-8px, 按钮 4-6px |

## 区块逐层分析

### 1. 导航栏 Header
- **位置**: 固定顶部 (sticky/fixed)
- **高度**: 约 70-80px
- **背景**: 白色 + 底部细阴影或边框
- **内容**: Logo(左) + 导航菜单(右/居中)
- **导航项间距**: 约 24-32px (水平 gap)
- **Logo ↔ 导航**: flex space-between 布局
- **下拉菜单**: hover 展开，含图标+文字，每项约 padding 10-16px vertical

### 2. Hero 主视觉区
- **布局**: 左右双栏 (文字左 / 图片轮播右)
- **垂直内边距**: 约 80-120px (上下)
- **标题字号**: 约 48-56px, 粗体
- **副标题字号**: 约 16-18px, 灰色 (#666-#888)
- **标题与副标题间距**: 约 16-20px
- **副标题与 CTA 间距**: 约 24-32px
- **CTA 按钮**: 圆角胶囊形，黑底白字或黄绿底深字，padding 约 14px 28px
- **右侧图片轮播**: 多图网格/轮播展示，图片等比缩放

### 3. 产品摄影服务介绍区
- **背景**: 浅灰/米白 (#F5F5F5 或 #FAFAFA)
- **上边距**: 约 60-80px (与前区块距离)
- **下边距**: 约 60-80px
- **左右内边距**: 约 40-60px
- **标题**: 居中, 字号约 36-42px
- **副标题**: 居中, 灰色, 字号约 16-18px
- **标题↔副标题间距**: 约 12-16px
- **副标题↔内容区**: 约 40-48px
- **内容布局**: 两栏 (文字描述 + 图片/卡片)
- **栏间距(gap)**: 约 30-40px
- **CTA 按钮位置**: 区域底部居中，距底部约 20-32px

### 4. 服装摄影展示区 (Clothing Photography)
- **背景**: 白色
- **上边距**: 约 60-80px
- **下边距**: 约 60-80px
- **标题**: 左对齐或居中, 约 32-40px
- **分类标签栏**: 横向排列, 标签间距约 12-16px, 标签内 padding 约 8px 16px
- **标签↔图片网格间距**: 约 24-32px
- **图片网格**: Masonry/瀑布流 或 Grid 布局
- **列数**: 3-4列 (1440px视口)
- **图片间距(gap)**: 约 12-16px
- **图片宽高比**: 约 500:563 (~0.89) 或 500:579 (~0.86)

### 5. 产品摄影展示区 (Product Photography)
- **结构同上区**
- **分类标签**: Shoes / Eyewear / Amazon / Home goods / Beauty / Supplements / Others
- **标签样式**: 文字链或 pill 形按钮
- **图片网格**: 同样 3-4列, gap 约 12-16px
- **与前区间距**: 约 60-80px (白色背景连续时可略小, ~40-60px)

### 6. 电商产品视频服务横幅 (Video CTA Banner)
- **布局**: 横向全宽或容器宽度, 左文右图或纯色背景+文字
- **背景**: 可能为深色或品牌色
- **上下内边距**: 约 50-72px
- **左右内边距**: 约 40-60px
- **文字与按钮横向排列**
- **两个 CTA**: "Get in touch" (实心) + "Learn more" (描边/空心)
- **按钮间距**: 约 12-16px
- **与前区间距**: 约 40-60px

### 7. AI 生活场景摄影介绍 (AI Lifestyle Intro)
- **布局类似视频横幅**
- **上下内边距**: 约 50-72px
- **CTA 按钮组合同上**

### 8. AI 时尚模特展示 (AI On-Model Gallery)
- **背景**: 白色或浅灰
- **上边距**: 约 60-80px
- **标题 + 副标题**: 居中, 间距约 12-16px
- **图片网格**: 3-4列, gap 约 12-16px
- **可能包含视频元素 (webm)**
- **图片尺寸多样** (500x465 至 500x742), 不规则瀑布流

### 9. AI 亚马逊生活场景 (AI Lifestyle for Amazon)
- **紧凑展示区**, 仅4张图
- **可能是 2x2 网格或 1x4 横排**
- **上边距**: 约 60-80px

### 10. 3D 产品渲染服务 (3D Rendering CTA)
- **类似视频/AI区的 CTA 横幅布局**
- **包含播放按钮图标**
- **上下内边距**: 约 50-72px
- **可能含两张大图 + 文字叠加**

### 11. 亚马逊引导横幅 ("Selling on Amazon?")
- **全宽背景图** (bg.webp)
- **叠加文字**: 标题 + 副标题 + CTA
- **垂直内边距**: 约 80-120px (较宽松)
- **文字居中**
- **CTA 按钮**: "Start now", 黄绿/亮色背景
- **与前区间距**: 0 (全宽无缝连接)

### 12. 工作流程 (How does it work?)
- **背景**: 白色或浅色
- **上边距**: 约 80-100px
- **标题 + 副标题**: 居中, 副标题灰色
- **步骤布局**: 4列水平排列 (桌面端)
- **每个步骤**: 图标(上) + 标题(中) + 描述(下)
- **步骤间距(gap)**: 约 24-32px
- **步骤内部**: 图标↔标题约 16px, 标题↔描述约 8-12px
- **步骤↔CTA**: 约 32-40px
- **下边距**: 约 80-100px

### 13. 为什么选择我们 (Why Choose Us)
- **背景**: 浅色 (#FAFAFA 左右)
- **上边距**: 约 60-80px
- **装饰性顶图**: content-top.webp
- **标题 + 副标题**: 居中
- **6个优势卡片**: 3列 x 2行 Grid 布局 (桌面端)
- **卡片间距(gap)**: 约 20-24px
- **卡片内边距**: 约 24-32px
- **卡片内**: 标题(粗) + 描述(灰色正文), 间距约 8-12px
- **亮点总结行**: 3个亮点 (Top Quality / Fast & Easy / All-in-One), 横向排列
- **装饰性底图**: content-footer.webp
- **CTA**: "Book a shoot"
- **下边距**: 约 60-80px

### 14. 合作品牌展示 (Brands We Work With)
- **背景**: 白色
- **上边距**: 约 40-60px
- **标题**: 居中
- **品牌 Logo 行**: 3-4行独立滚动轮播
- **每行 Logo 数量**: 约 15-20个
- **Logo 间距**: 约 24-32px (均匀分布)
- **行间距**: 约 16-24px
- **Logo 样式**: 灰度或半透明, hover 可能变色
- **下边距**: 约 40-60px

### 15. SEO 文本区 (What we do)
- **背景**: 白色或极浅灰
- **上边距**: 约 40-60px

- **文本内容**: 最大宽度约 800px (阅读最佳宽度)
- **文本行高**: 约 1.6-1.8
- **段落间距**: 约 16-20px

### 16. FAQ 折叠区 (FAQ)
- **背景**: 可能是浅灰 #F5F5F5 或白色

- **标题**: 居中, "Frequently asked questions"
- **FAQ 项目**: 手风琴/折叠组件
- **问题间距**: 约 8-12px (项目之间)
- **问题内边距**: 约 16-24px

### 17. 页脚 Footer
- **背景**: 深色 (#1a1a2e 或黑色渐变)

- **版权信息**: 最底部, 更小的字号, 更浅的灰

## 通用间距规律总结

### 区块间垂直间距 (Section Spacing)
```
标准区间距:   60 - 80px    (最常用)
紧密区间距:   40 - 60px    (同类内容连续时)
强调区间距:   80 - 120px   (主要视觉转换处)
CTA横幅间距:  50 - 72px
```

### 容器内边距 (Container Padding)
```
左右 padding:  20px (mobile) → 40-60px (desktop)
上下 padding:  60-80px (标准区块)
              80-120px (Hero / 大型视觉区)
              40-60px (紧凑区)
```

### 元素级间距 (Element Spacing)
```
标题 ↔ 副标题:      12 - 16px
副标题 ↔ 内容:     24 - 48px (视重要性)
段落之间:           16 - 20px
按钮组间距:         12 - 16px
Grid gap (图片):     12 - 16px
Grid gap (卡片):     20 - 24px
步骤/卡片内部:       8 - 16px
```

### 字号层级 (Typography Scale)
```
H1 (Hero 主标题):    48 - 56px
H2 (区块标题):      32 - 42px
H3 (子标题):        24 - 28px
H4 (卡片标题):      18 - 22px
Body 正文:           16px
Small (辅助文字):    14px
Caption (说明):      12 - 13px
```

## 设计节奏模式 (Visual Rhythm Pattern)

```
[Header]
    ↓ 0px (fixed)
[Hero] ← 120px padding, 密集信息
    ↓ 80px
[服务介绍] ← 浅色背景切换, 中等密度
    ↓ 80px
[服装摄影] ← 白色背景, 图片密集区
    ↓ 60px
[产品摄影] ← 白色背景延续, 紧凑过渡
    ↓ 60px
[视频 CTA] ← 横幅式, 节奏变化 ★
    ↓ 60px
[AI 介绍] ← 另一个横幅, 连续节奏变化
    ↓ 80px
[AI 模特展示] ← 回到图片密集
    ↓ 80px
[AI Amazon] ← 紧凑 4图
    ↓ 60px
[3D 渲染 CTA] ← 横幅节奏变化 ★
    ↓ 0px (全宽无缝)
[Amazon 引导] ← 全宽背景图, 强视觉冲击 ★★★
    ↓ 80px
[工作流程] ← 信息型, 步骤化
    ↓ 80px
[Why Us] ← 卡片网格 + 装饰图
    ↓ 60px
[合作品牌] ← 轻量滚动动画
    ↓ 60px
[SEO 文本] ← 纯文字, 收窄宽度
    ↓ 60px
[FAQ] ← 折叠交互
    ↓ 80px
[Footer] ← 深色收尾
```

**关键设计规律:**
1. **图片密集区** 和 **文字/CTA横幅区** 交替出现，形成呼吸感
2. 每 2-3 个密集区后接一个 CTA 横幅作为节奏断点
3. 背景色在白色/浅灰之间交替（但不是每个区块都切换）
4. 全宽背景图区域 (Amazon引导) 作为整页的高潮点
5. 从上到下：信息密度呈"高→中→高→中→低"递减趋势
